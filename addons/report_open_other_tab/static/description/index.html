Open Report with Other Tab

Overview

This module enhances the Odoo reporting experience by enabling developers to open reports in a new tab for PDF or HTML. This functionality allows developers to directly preview changes made to reports simply by refreshing the report's tab. switch between PDF and HTML formats directly through the URL, eliminating the need for additional steps and saving valuable development time.

Key Features

  • Open reports (PDF/HTML) in a new browser tab.
  • Directly preview report changes with a simple page reload.
  • Switch report formats between PDF and HTML via URL parameters.
  • Streamline the report development and debugging workflow.

Dev Tip

Use --dev=xml to directly reload arch from xml file instead of database. Change in xml and reload the page to view changes.

How to enable

Go to Settings > Technical > Action > Reports
